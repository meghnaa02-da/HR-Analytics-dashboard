# HR Employee Distribution Dashboard

An interactive **Power BI** dashboard built using **DAX** to analyze workforce composition, distribution, and department-level trends across an organization.

## 📌 Project Overview

This project explores employee data to understand how a workforce is distributed across gender, race, location, age group, and department — along with trends in employee turnover and length of employment. The dashboard is designed to give HR teams a quick, visual read on workforce diversity and department health.

## 🎯 Objective

- Analyze workforce composition across demographics (gender, race, age group)
- Track employee distribution across locations/states
- Monitor employee turnover and average length of employment
- Identify department-level patterns in headcount and termination rate

## 🛠️ Tools & Technologies

| Tool | Purpose |
|------|---------|
| **Power BI** | Dashboard design and data visualization |
| **DAX** | Calculated measures (e.g., average length of employment, termination rate, net change %) |

## 📊 Dashboard Pages

### 1. Workforce Overview
- **KPI Card** — Average Length of Employment
- **Column Chart** — Employee count by Gender
- **Donut Chart** — Employee distribution by Location
- **Line Chart** — Net change % in headcount by Year
- **Map** — Total employees by Location/State
- **Column Chart** — Employee count by Race

### 2. Department Insights
- **Column Chart** — Employee count by Age Group
- **Clustered Column Chart** — Age Group distribution by Gender
- **Table** — Department-wise Termination Rate and Total Employee Count
- **Clustered Column Chart** — Department-wise headcount by Gender

## 🔍 Key Insights

- Workforce composition varies across gender, race, and age groups
- Certain locations/states account for a larger share of total headcount
- Departments differ notably in termination rate, highlighting where retention needs attention
- Year-over-year net change trends show periods of workforce growth or contraction.
